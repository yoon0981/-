# PWA (웹앱) - iPhone에서 '앱처럼' 쓰는 가장 쉬운 방법

이 폴더는 **Safari에서 열고 → '홈 화면에 추가'** 하면 앱처럼 실행되는 웹앱(PWA)입니다.

## 핵심 장점
- Mac/Xcode 없이도 가능
- 계정/회원가입 없이 사용 가능(단, 웹페이지를 올려둘 '장소'는 필요)

## 사용 방법(추천)
1) 이 폴더를 웹에 올립니다. (예: GitHub Pages / Netlify Drop / Cloudflare Pages 등)
2) iPhone Safari로 주소를 엽니다.
3) 공유 버튼(사각형+화살표) → **홈 화면에 추가**
4) 이제 아이콘이 생기고, 터치하면 앱처럼 열립니다.
5) 한 번 열어두면 오프라인에서도 캐시로 동작합니다(서비스워커).
## 지도 버튼
- 각 항목의 **T맵** 버튼을 누르면 티맵 앱으로 이동합니다.
- 티맵이 설치되어 있지 않으면 App Store(티맵) 페이지로 안내합니다.


## 로컬에서만 쓰고 싶다면
- 같은 와이파이 안의 PC/Mac에서 간단한 로컬 서버를 띄우고(i.e. python -m http.server),
  iPhone Safari에서 해당 IP로 접속해 '홈 화면에 추가'를 하세요.
